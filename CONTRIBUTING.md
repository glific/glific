Thanks for your intrest. To start contributing in this repo. 

To get started:

1. Clone this Repo
2. Setup the project by following the steps in the [Readme.md](https://github.com/glific/glific#readme)
3. Pick one issue ([beginner level issues](https://github.com/glific/glific-frontend/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22good%20first%20issue%22))
4. Whenever you create a PR, make sure:
     - All CI checks are passing 
     - All CodeRabbit review comments have been properly addressed before requesting a review.
5. You can reach out to us on our [discord server](https://discord.com/channels/717975833226248303/1468837556966916289) in case of any queston/suggestions.
